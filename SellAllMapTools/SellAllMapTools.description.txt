The mod adds five new tools that allow player to sell buildings all over the map.

The first set of tools consists of two tools:
- Building to Sell Picker Tool
- Sell All Picked Buildings Tool

The "Building to Sell Picker Tool" allows the player to select the type of building to sell.
Use mouse wheel to select the previous selected building.

The "Sell All Picked Buildings Tool" allows the player to sell all buildings of the selected type on the map.

The "Sell All Picked Buildings Tool" allows the player to sell all buildings of the selected building group.
The player can change the operating mode of the tool by mouse wheel.



The second set of tools consists of two tools:
- Building Category to Sell Picker Tool
- Sell All Buildings of Category Tool

The "Building Category to Sell Picker Tool" allows the player to select the category of building to sell.
Use mouse wheel to select the previous selected category.

The "Sell All Buildings of Category Tool" allows the player to sell all buildings of the selected category on the map.
Use mouse wheel to select the previous selected category.



The fifth tool is Sell All Buildings Tool.

The "Sell All Buildings Tool" allows the player to sell all buildings on the map.


When using Picker Tools, buildings that can be selled are highlighted in yellow.


Warning. None of the three tools will sell the Headquarters.



The mod will not conflict with any other mods as all files in this mod are new.

New tools will appear in the new game and in saves.
