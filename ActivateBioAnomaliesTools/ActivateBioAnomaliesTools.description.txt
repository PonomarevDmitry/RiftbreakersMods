The mod adds tools for activating BioAnomalies.

Activate BioAnomaly Tool allows player to activate selected BioAnomalies.

Activate All BioAnomalies Tool allows player to activate All BioAnomalies on the map.

Tools only work with BioAnomalies.

------------------------------------------------------------------------------------------------------

The mod will not conflict with any other mods as all files in this mod are new.

New buildings and tools will appear in the new game and in saves.

Localizations:
- English
- Russian

Updated to DLC3.
