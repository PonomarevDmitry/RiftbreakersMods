The mod adds a plant icon above the cultivator in build mode.

The mod changes the plant selection icon to the current plant icon in the cultivator.

The mod also adds Production category to the cultivator's Building Info.
Production category displays the name of the plant and the resources that can be collected from the plant.

Plant icon and Production will be shown in new game and saves.
