Instrukcja tworzenia grafu animacji : 
1. skopiuj animation_graph_template.ag. Zmień nazwę skopiowanego pliku, otwórz go i wpisz w polu name odpowiednią nazwę.
2. dodaj do entita komponent:
	AnimationGraphDesc
	{
		animation_graph_name "my_animation_graph_name"
	}	
3. otworz entita w model edytorze, wybierz zakladke animation graph
4. stwórz swój graf animacji. :)
