The mod adds three sets of tools that allow the player to replace gates with others types.

------------------------------------------------------------------------------------------------------

The first set of tools consists of three tools:
- Replace Selected Gates with Energy Gates
- Replace Selected Gates with Crystal Gates
- Replace Selected Gates with Wall Gates
- Replace Selected Gates with Vine Walls

If the gates are not available for construction, warning will be shown.

------------------------------------------------------------------------------------------------------

The second set of tools consists of tool "Gate Type Picker Tool" and tool "Replace All Selected Gates with Picked Gate Type".

The "Gate Type Picker Tool" tool allows the player to select the type of gate that other gates will be replaced with when using the "Replace All Selected Gates with Picked Gate Type" tool.
Use mouse wheel to select the previous selected gate.

The "Replace All Selected Gates with Picked Gate Type" tool allows the player to select an area, and all gates in that area will be replaced by gates of the selected type.

It is necessary that the type of gate to replace is selected when using the "Replace All Selected Gates with Picked Gate Type" tool.
A warning will be displayed if the gate type is not selected.

------------------------------------------------------------------------------------------------------

The third set of tools consists of four tools:
- Gate Type 1 Picker Tool
- Gate Type 2 Picker Tool
- Replace Selected Gates Type 1 with Type 2
- Replace Selected Gates Type 2 with Type 1

Tools "Gate Type 1 Picker Tool" and "Gate Type 2 Picker Tool" allow the player to select gate Type 1 and gate Type 2.
Use mouse wheel to select the previous selected gate.

The "Replace Selected Gates Type 1 with Type 2" tool allows the player to select an area, and all Type 1 gates in that area will be replaced by Type 2 gates.

On the contrary, the "Replace Selected Gates Type 2 with Type 1" tool allows the player to select an area, and all Type 2 gates in that area will be replaced by Type 1 gates.

It is necessary that gate Type 1 and gate Type 2 are selected when using tools "Replace Selected Gates Type 1 with Type 2" and "Replace Selected Gates Type 2 with Type 1".
A warning will be displayed if gate Type 1 or gate Type 2 is not selected.

------------------------------------------------------------------------------------------------------

The gate is replaced with the same level or with a lower available level.

------------------------------------------------------------------------------------------------------

The mod will not conflict with any other mods as all files in this mod are new.

New tools will appear in the new game and in saves.
