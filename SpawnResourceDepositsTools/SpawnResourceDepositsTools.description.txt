The mod adds seven tools to spawn resource deposits.

Spawn Carbonium Ore Deposit Tool allows player to spawn Carbonium Ore Deposit from player's Carbonium storage.

Spawn Ironium Ore Deposit Tool allows player to spawn Ironium Ore Deposit from player's Ironium storage.

Spawn Cobalt Ore Deposit Tool allows player to spawn Cobalt Ore Deposit from player's Cobalt storage.

Spawn Palladium Ore Deposit Tool allows player to spawn Palladium Ore Deposit from player's Palladium storage.

Spawn Titanium Ore Deposit Tool allows player to spawn Titanium Ore Deposit from player's Titanium storage.

Spawn Uranium Ore Deposit from Crude Uranium Tool allows player to spawn Uranium Ore Deposit from player's Crude Uranium storage.

Spawn Uranium Ore Deposit from Uranium Tool allows player to spawn Uranium Ore Deposit from player's Uranium storage.

Use mouse wheel to change the size of the resource deposit.

The amount to increase or decrease the value increases as player use the mouse wheel quickly and for a long time.



The mod will not conflict with any other mods as all files in this mod are new.

New tools will appear in the new game and in saves.
