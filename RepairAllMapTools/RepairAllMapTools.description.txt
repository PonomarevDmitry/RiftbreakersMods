The mod adds five new tools that allow player to repair buildings all over the map.

The first set of tools consists of two tools:
- Building to Repair Picker Tool
- Repair All Picked Buildings Tool

The "Building to Repair Picker Tool" allows the player to select the type of building to repair.
Use mouse wheel to select the previous selected building.

The "Repair All Picked Buildings Tool" allows the player to repair all buildings of the selected type on the map.



The second set of tools consists of two tools:
- Building Category to Repair Picker Tool
- Repair All Buildings of Category Tool

The "Building Category to Repair Picker Tool" allows the player to select the category of building to repair.
Use mouse wheel to select the previous selected category.

The "Repair All Buildings of Category Tool" allows the player to repair all buildings of the selected category on the map.
Use mouse wheel to select the previous selected category.



The fifth tool is Repair All Buildings Tool.

The "Repair All Buildings Tool" allows the player to repair all buildings on the map.



The mod will not conflict with any other mods as all files in this mod are new.

New tools will appear in the new game and in saves.

Localizations:
