The mod contains lua-script to remove "Scanner Drones" research tree from the game.

Steps to delete "Scanner Drones" research tree:
1. Delete mod "Scanner Drones", install this mod "Scanner Drones - Research Tree Remover"
2. Restart game
3. Load save
4. Create new save without "Scanner Drones" research tree
5. Delete mod "Scanner Drones - Research Tree Remover".



The mod will not conflict with any other mods as all files in this mod are new.

Updated to DLC3.
